# UI Guidelines

## Typography

- Headings: Use `font-black uppercase tracking-tighter`.
- Labels: Use `text-[10px] font-bold uppercase tracking-widest text-slate-500`.

## Colors

- Background: Deep Slate (`bg-[#020617]`).
- Accent: Blue (`text-blue-500`) for navigation, Red (`text-red-500`) for Battle.

## Spacing

- Use `gap-6` for grids and `p-4` or `p-8` for containers.
